# Iowa State University Bookstore

The proposed solution is to redesign the Iowa State University Bookstore’s textbook search interface so that it better matches what users expect and what is commonly seen on other university bookstore websites. The goal of this redesign is to make the interface clearer, easier to navigate, and more usable overall, without changing what the system is meant to do.

This project focuses on creating an interface that makes the textbook search feature more visible and easier to find, helping students quickly understand where to start when looking for course materials. The updated experience would cut down on unnecessary steps, use clearer labels, and apply a stronger visual hierarchy to guide users through the process. Improving the overall structure and flow of the interface would make it easier for users to locate required textbooks and view pricing information.

Rather than adding new features, the proposed solution centers on improving how existing features are organized and presented. These changes aim to reduce confusion, lower frustration, and encourage students to use the bookstore website more confidently as a dependable source for textbook information.
